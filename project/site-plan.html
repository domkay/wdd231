Tech Gadget Hub - Site Plan

Site Name

Tech Gadget Hub

Reason for Selection: The name "Tech Gadget Hub" clearly reflects the purpose of the site, which is to serve as a central platform for tech enthusiasts to explore, compare, and review the latest gadgets and tech products. It is simple, memorable, and directly related to the content and functionality of the website.

Site Purpose

The Tech Gadget Hub website aims to provide a comprehensive platform for tech enthusiasts to:

  • Explore the latest gadgets and tech products.
  • Compare products based on specifications, prices, and user ratings.
  • Submit reviews and feedback on products.
  • Save preferred products to a wishlist for future reference.

The site will serve as a one-stop destination for users looking to make informed decisions about tech purchases.

Color Schema

  • Primary Color: #1A73E8 (Blue) - Used for headings, call-to-action buttons, and navigation bar.
  • Secondary Color: #34A853 (Green) - Used for accent elements, success messages, and interactive features like the "Add to Wishlist" button.
  • Background Color: #FFFFFF (White) - Used for the main background to ensure a clean and modern look.
  • Text Color: #202124 (Dark Gray) - Used for body text and general content.

Typography

  • Primary Font: Roboto - Used for headings, body text, and navigation.
  • Secondary Font: Open Sans - Used for subheadings and special sections.
  • Accent Font: Montserrat - Used for call-to-action buttons and highlighted text.
